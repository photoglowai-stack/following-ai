# Denylisted Tools

Ces outils ne sont pas intégrés car ils peuvent violer les conditions d'utilisation des plateformes, contourner des protections ou exposer des données privées.

The MVP must not integrate tools that require private sessions, session cookies, private user tokens, login bypasses, anti-bot bypasses, or scraping of private accounts.

## Denylist

- Toutatis / Instagram session tools
- Osintgram-like tools
- TikTok session scrapers
- LinkedIn scrapers non officiels
- Tout outil nécessitant `sessionid`, cookies privés, tokens utilisateurs ou contournement anti-bot

## Policy

- No private platform sessions.
- No account takeover flows.
- No aggressive password reset automation.
- No captcha, paywall, login, or rate-limit bypass.
- No scraping private profiles or private content.
- No storage by default.
